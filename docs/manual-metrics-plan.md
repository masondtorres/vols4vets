# Manual Metrics Plan

Vols4Vets should stay useful without adding tracking pixels, cookies, backend analytics or user accounts in this phase.

## Weekly Metrics Mason/Jen Can Track

- Vercel visits.
- Most visited pages.
- Google Search Console indexed pages.
- Google Search Console search queries.
- Number of correction requests.
- Number of broken links fixed.
- Number of local details verified.
- Top Find My Next Step paths if safe aggregate tracking is added later.
- Toolkit pages added or refreshed.
- Outreach emails sent.
- Replies from official offices.
- Backlinks earned from legitimate public references.

## Ground Rules

- Do not collect private case details for metrics.
- Do not add analytics code in this pass.
- Do not add tracking pixels.
- Do not add cookies.
- Do not rank resources by payment, sponsorship or private deals.
- Keep measurement focused on whether veterans can find official starting points faster.

## Weekly Review Rhythm

1. Check Vercel traffic and top pages.
2. Check Search Console indexing and queries.
3. Review correction requests and broken links.
4. Update the monthly log if a public-facing change shipped.
5. Pick one page to improve based on user need, not vanity traffic.
