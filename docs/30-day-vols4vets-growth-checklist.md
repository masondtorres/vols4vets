# 30-Day Vols4Vets Growth Checklist

## 1. Veteran Needs

- This week: Review Find My Next Step paths for clarity and safety.
- This month: Add one missing checklist based on real user questions.
- Do not do yet: Collect personal stories or private case details.
- Devil's advocate risk: More content can create more confusion if the first step is not obvious.

## 2. Content

- This week: Refresh the three highest-use toolkit pages.
- This month: Add county-by-county pages only when there is a useful local angle.
- Do not do yet: Publish thin pages for every city just to chase keywords.
- Devil's advocate risk: Local SEO can become keyword stuffing if pages repeat the same copy.

## 3. Community

- This week: Keep community guidelines public but do not launch a forum.
- This month: Collect safe correction patterns from public feedback.
- Do not do yet: Add comments, accounts or user profiles.
- Devil's advocate risk: Community features can attract private records, claims advice and moderation burden.

## 4. Tech

- This week: Run the static validation script after each content pass.
- This month: Expand validation for search coverage and stale dates.
- Do not do yet: Add a backend or dependencies without a clear maintenance owner.
- Devil's advocate risk: Tooling can become harder to maintain than the site itself.

## 5. Visibility

- This week: Submit updated sitemap in Search Console after deploy.
- This month: Track long-tail queries and improve pages that already match real searches.
- Do not do yet: Buy backlinks, fake reviews or directory spam.
- Devil's advocate risk: Visibility work can damage trust if it looks like hype.

## 6. Partnerships

- This week: Avoid partnership language unless there is a real written relationship.
- This month: Build a private list of official offices to verify manually.
- Do not do yet: Add sponsor slots, paid placement or implied endorsements.
- Devil's advocate risk: Even friendly outreach can sound like affiliation if wording is loose.

## 7. Operations

- This week: Review correction requests and broken links.
- This month: Confirm changed hours, phone numbers and appointment rules from official sources.
- Do not do yet: Promise live monitoring or emergency response.
- Devil's advocate risk: A stale local page can be worse than no local page.

## 8. Sustainability

- This week: Keep the content plain enough for one person to maintain.
- This month: Decide which pages get monthly, quarterly or as-needed review.
- Do not do yet: Add donations, payments or recurring obligations.
- Devil's advocate risk: Growth without maintenance makes veterans do extra work.
