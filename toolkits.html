Use these plain checklists to prepare for official websites, verified offices and accredited help.

Fast answer: Pick the checklist that matches the problem, print it if useful and use the official links before sharing private records.

How to use these toolkits

These pages help you organize the first move. They do not replace VA, legal, medical, crisis or accredited claims help.

Crisis warning: If there is immediate danger, call 911. For veteran crisis support, call 988 then press 1 or text 838255.
